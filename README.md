# OIBSIP_python_taskno3
Objective:

 Created a command-line weather app in Python that fetches and displays current weather data for a user-specified location (e.g., city or ZIP code) using a weather API. Show basic information such as temperature, humidity, and weather conditions.

Steps Performed:

   1.Imported the required Python modules for HTTP requests and JSON processing.
   
   2.Accepted the location name as user input.
   
   3.Sent a request to the weather service and retrieved live weather data.
   
   4.Extracted key weather details such as temperature, humidity, wind speed, pressure, visibility, and weather conditions.
   
   5.Displayed the weather information in a user-friendly format.
   
   6.Implemented error handling for invalid locations, internet connectivity issues, and incorrect responses.
   
Tools Used:

   Programming Language: Python
          
   IDE: Visual Studio Code
          
   Libraries: urllib, json
          
   Weather Service: wttr.in API
          
Outcome:

Successfully developed a command-line Weather Application that provides real-time weather updates for any location. The application delivers accurate weather information, validates user input, and handles errors effectively, demonstrating API integration and data processing using Python.
